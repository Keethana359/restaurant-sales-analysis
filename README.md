# Retail Sales Analysis Project

## 📊 Overview
This project analyzes retail sales data to uncover business insights related to product performance, customer behavior, and revenue trends.

## 🛠️ Tools Used
- SQL
- Python (Pandas, NumPy, Matplotlib)
- Power BI

## 🔍 Key Analysis
- Sales trends over time
- Top-performing products
- Regional performance
- Customer segmentation

## 📈 Key Insights
- Top 5 products contribute ~40% of revenue
- Sales peak during holiday season
- West region shows highest growth
